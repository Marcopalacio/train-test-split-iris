## Descripción
Este proyecto aplica una partición train/test a un dataset para entrenamiento de modelos de inteligencia artificial.

## Dataset utilizado
- Nombre: Iris Dataset
- Fuente original: https://github.com/mwaskom/seaborn-data

## Configuración del split
- Proporción: 80% entrenamiento / 20% prueba
- Semilla: 42

## Archivos generados
Carpeta `/splits`:
- X_train.csv
- X_test.csv
- y_train.csv
- y_test.csv
